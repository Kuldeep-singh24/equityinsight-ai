# EquityInsight AI – Financial News Research & Q&A System

## Overview
EquityInsight AI is an intelligent financial news research and Q&A system that helps users analyze financial articles and extract insights using LLMs and vector search.

## Features
- Financial news analysis
- Question answering on news articles
- Semantic search using FAISS
- LLM-powered responses
- Streamlit-based interface

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Hugging Face
- Groq API

## Installation

```bash
pip install -r requirements.txt
streamlit run main.py
```
